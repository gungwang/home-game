A cyberpunk-themed 2D side-scrolling web shooter built with React + Phaser 3 + TypeScript.

The player controls a dragon flying over Manhattan, battling mutant farm animals, weaving through skyscrapers, and watching a YouTube music video after each checkpoint. A professional resume is displayed upon completion.

Features

  • Cyberpunk visuals вҖ” Dark backgrounds, neon palette, weather system (rain, snow, lightning), day/night cycle (Day вҶ’ Sunset вҶ’ Night вҶ’ Sunrise)
  • Dual weapon system вҖ” Dragon breath fireballs (unlimited) + Dragon spike missiles (limited), with weapon upgrades
  • YouTube video integration вҖ” A YouTube MV plays on building surfaces after each checkpoint, earning bonus score
  • Boss battle вҖ” Boss fight triggers after checkpoint 9; defeat the boss to win
  • Resume showcase вҖ” Professional resume and leaderboard displayed on game completion
  • Keyboard controls вҖ” Arrow keys / WASD to move,
    Left mouse button вҮ’ Fire a fireball
    Right mouse button вҮ’ Fire a missile

Tech Stack

  • Frontend: React 19 + TypeScript
  • Game Engine: Phaser 3 (Arcade Physics)
  • Build Tool: Vite
  • Styling: Tailwind CSS
  • Video Player: react-youtube

Quick Start

npm install
npm run dev

Run Local Server

  1. Install dependencies: npm install
  2. Start the local development server: npm run dev
  3. Open: http://localhost:5173

If port 5173 is already in use, Vite will automatically choose another available port and print the actual local URL in the terminal.
