# 飞龙大战纽约 (Dragon vs New York)

网页2D横版游戏，以赛博朋克2077的风格和主题为核心。结合Youtube频道100多个音乐视频，玩家完成一个小关卡后播放一个视频。打怪和观看视频都能获得积分。

## 游戏设计

### 地图
- 纽约-曼哈顿城，从中央公园到自由女神像
- 50-100高楼大厦，包含纽约标志性建筑

### 故事背景
各种可爱的家畜动物（鸡、鸭、猪、羊、牛）突变成妖怪，占领纽约城。神龙降临，在空中飞行，消灭大厦上的妖怪。

### 视觉风格
- **配色**：黑色、白色、淡红、淡黄、淡蓝、淡绿，可有透明过渡效果
- **天气系统**：天气变化、四季、白天黑夜循环
- **角色**：卡通形象，可爱、简单
- **背景**：深色背景，不恐怖，轻松休闲，主题优美简洁

### 战斗系统
- **龙息火球**：普通攻击，无限弹药
- **龙刺导弹**：重击，有限弹药
- 击杀妖怪得分，妖怪分级，得分不同

### 视频系统
- 100多个Youtube MV视频
- 每过一小关，在大楼表面播放一首视频
- 观看完视频可得分，杀怪得分和视频得分平衡
- 每次刷新页面，视频随机排列

### 游戏参数
- 通关时长：10-30分钟
- 操作方式：键盘和鼠标
- 通关后显示个人信息（工作简历）

### 素材
如需游戏素材，从网络搜索下载。
